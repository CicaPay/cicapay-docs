Welcome to the **CicaPay API** documentation.

CicaPay is a crypto payment gateway that allows businesses to **accept payments, create invoices, and send payouts using cryptocurrency**.

With the CicaPay API, you can:

- Accept crypto payments from customers
- Generate payment requests
- Create and manage invoices
- Send payouts to external wallets
- Receive real-time payment notifications via webhooks
