## Overview

Building an automated payment system is hard enough — having to learn blockchain, manage wallets just to *test* it makes it even harder... We built our own faucet so you can skip all of that.

Get free testnet tokens sent directly to your address in seconds, no crypto knowledge required.

<Callout title="Test Environment Only">
Faucet tokens are on **BSC Testnet** and carry no real-world value. They can only be used to test [Payment Requests](/docs/payment-request) and [Invoices](/docs/invoice) in the sandbox environment.
</Callout>

## FAQ

<Accordions>

<Accordion title="What is a faucet?">
A faucet is a tool that distributes small amounts of testnet tokens for free. Testnet tokens have no real monetary value — they exist solely for developers to test integrations on a blockchain test environment without spending real money.
</Accordion>

<Accordion title="How does the faucet work?">
Enter your CicaPay public API key (any active key from your dashboard), the destination wallet address, and the amount you want to receive. Our system validates your key, checks the rate limit, and dispatches the tokens on BSC Testnet. The transaction is confirmed on-chain within a few seconds to a couple of minutes depending on network conditions.
</Accordion>

<Accordion title="How many tokens can I request?">
Each request can be between `> 0` and `1 BTT`. You can make up to **10 requests per hour**. Once the rate limit is reached, you will need to wait before making another request.
</Accordion>

<Accordion title="How long does it take for the testnet tokens to arrive?">
Tokens are typically delivered within **a few seconds to 2 minutes** after a successful request. BSC Testnet block times are fast, but occasional network congestion may cause slight delays. You can check the transaction status using the `tx_hash` shown in the history panel.
</Accordion>
